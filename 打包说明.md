# 坤展成-中控多播放器 - 打包说明

## PyInstaller 打包指南

### 准备工作

1. 安装 PyInstaller
```bash
pip install pyinstaller
```

2. 确保所有依赖已安装
```bash
pip install -r requirements.txt
```

### 打包命令

#### 方式一：单文件打包（推荐）

```batch
pyinstaller --name="坤展成中控播放器" ^
            --windowed ^
            --onefile ^
            --icon=icon.ico ^
            --add-data=".;." ^
            --hidden-import=PyQt5 ^
            --hidden-import=PyQt5.QtCore ^
            --hidden-import=PyQt5.QtGui ^
            --hidden-import=PyQt5.QtWidgets ^
            --hidden-import=PyQt5.QtMultimedia ^
            --hidden-import=PyQt5.QtNetwork ^
            --hidden-import=vlc ^
            --hidden-import=serial ^
            --hidden-import=serial.tools.list_ports ^
            --collect-all=vlc ^
            main.py
```

#### 方式二：文件夹打包（更稳定）

```batch
pyinstaller --name="坤展成中控播放器" ^
            --windowed ^
            --onedir ^
            --icon=icon.ico ^
            --add-data=".;." ^
            --hidden-import=PyQt5 ^
            --hidden-import=PyQt5.QtCore ^
            --hidden-import=PyQt5.QtGui ^
            --hidden-import=PyQt5.QtWidgets ^
            --hidden-import=PyQt5.QtMultimedia ^
            --hidden-import=PyQt5.QtNetwork ^
            --hidden-import=vlc ^
            --hidden-import=serial ^
            --hidden-import=serial.tools.list_ports ^
            --collect-all=vlc ^
            main.py
```

### Windows 打包脚本 (build.bat)

创建 `build.bat` 文件：

```batch
@echo off
chcp 65001 >nul
echo ==========================================
echo   坤展成-中控多窗口播放器 打包工具
echo ==========================================
echo.

echo [1/4] 检查依赖...
python -c "import PyQt5" 2>nul || (echo 错误: 请先安装 PyQt5 && pause && exit /b 1)

echo [2/4] 安装 PyInstaller...
pip install pyinstaller -q

echo [3/4] 开始打包...
pyinstaller --name="坤展成中控播放器" ^
            --windowed ^
            --onefile ^
            --hidden-import=PyQt5 ^
            --hidden-import=PyQt5.QtCore ^
            --hidden-import=PyQt5.QtGui ^
            --hidden-import=PyQt5.QtWidgets ^
            --hidden-import=PyQt5.QtMultimedia ^
            --hidden-import=PyQt5.QtNetwork ^
            --hidden-import=vlc ^
            --hidden-import=serial ^
            --hidden-import=serial.tools.list_ports ^
            --collect-all=vlc ^
            main.py

echo [4/4] 打包完成!
echo.
echo 输出目录: dist\
echo 可执行文件: dist\坤展成中控播放器.exe
echo.
pause
```

### 打包后处理

1. **VLC 库文件**
   如果使用 VLC 播放器，需要将 VLC 安装目录下的 `libvlc.dll` 和 `libvlccore.dll` 复制到 exe 同目录下，或将 VLC 安装到系统路径。

2. **运行测试**
   在 `dist` 目录下运行生成的 exe 文件，确保所有功能正常。

### 常见问题

#### Q1: 打包后运行报错 "No module named 'xxx'"
**解决**: 添加 `--hidden-import=xxx` 参数

#### Q2: VLC 视频无法播放
**解决**: 
1. 确保系统已安装 VLC 播放器
2. 或将 VLC 的 DLL 文件复制到 exe 目录

#### Q3: 串口功能不可用
**解决**: 安装 pyserial 库

#### Q4: 打包后界面显示异常
**解决**: 添加 `--add-binary` 或 `--add-data` 参数复制必要的资源文件

### Linux/Mac 打包（可选）

Linux 下使用 PyInstaller 打包：
```bash
pyinstaller --name="kunplayer" \
            --windowed \
            --onefile \
            main.py
```

## 注意事项

1. **图标文件**: 如果需要自定义图标，准备 256x256 的 .ico 文件
2. **权限**: 打包和运行时需要管理员权限（用于全局快捷键）
3. **杀毒软件**: 某些杀毒软件可能误报，请添加信任
