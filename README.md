# 坤展成-中控多窗口播放器

## 软件简介

坤展成-中控多窗口播放器是一款专业的展厅多媒体控制软件，支持多窗口视频播放、集中控制、远程指令等功能。

## 主要功能

### 1. 多窗口视频播放
- 支持同时打开4个独立视频窗口
- 支持视频文件（MP4/AVI/MKV/MOV/WMV等）
- 支持图片文件（JPG/PNG/BMP）
- 无边框可移动窗口
- 窗口锁定/解锁功能

### 2. 播放控制
- 播放、暂停、停止、重播
- 视频切换（上一个/下一个）
- 独立音量控制
- 全屏播放

### 3. 通信控制
- UDP广播控制（端口8880）
- TCP广播控制（端口8881）
- 各窗口独立控制端口
- 串口通信支持

### 4. 指令格式
```
windowid_command_param
或
windowid:command:param
```

支持的命令：
- `play` - 播放
- `pause` - 暂停
- `stop` - 停止
- `replay` - 重播
- `next` - 下一个
- `prev` - 上一个
- `mute` - 静音
- `unmute` - 取消静音
- `all` - 全部播放

### 5. 全局快捷键
- `PageUp` - 呼出/隐藏主窗口+置顶
- `PageDown` - 窗口锁定/解锁
- `V` - 当前窗口锁定/解锁
- `F/Enter` - 全屏切换
- `ESC` - 退出全屏

## 安装运行

### 方式一：直接运行Python脚本

1. 安装Python 3.8+
2. 安装依赖：
```bash
pip install -r requirements.txt
```
3. 运行程序：
```bash
python main.py
```
或双击 `start.bat`

### 方式二：打包为EXE运行

1. 双击运行 `build.bat`
2. 等待打包完成
3. 在 `dist\坤展成中控播放器.exe` 找到生成的可执行文件

## 使用说明

### 1. 打开视频窗口
1. 选择窗口编号（1-4）
2. 设置窗口位置和大小，或点击"全屏铺满桌面"
3. 点击"打开窗口"按钮

### 2. 添加媒体文件
1. 点击"添加视频"按钮
2. 选择视频或图片文件
3. 文件会添加到列表中

### 3. 播放控制
- 选择要播放的媒体文件，双击或点击播放按钮
- 使用播放控制按钮进行暂停、停止等操作
- 拖动音量滑块调节音量

### 4. 远程控制
- 通过UDP/TCP发送指令控制
- 指令格式：`窗口号_命令` 或 `窗口号:命令`
- 示例：`1_play` 表示让窗口1播放

### 5. 授权激活
- 首次运行有30天试用期
- 点击"激活授权"按钮
- 输入机器码，联系经销商获取授权码
- 完成激活
